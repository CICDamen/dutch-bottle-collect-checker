Favicon Generator

To generate ICO favicon:

  1. Right-click on the bottle icon above and "Save image as" SVG
  2. Visit https://favicon.io/favicon-converter/ or similar favicon generator
  3. Upload the SVG file
  4. Download the generated favicon.ico
  5. Replace public/favicon.ico with the new file

Or use the existing bottle-icon.svg:

The bottle-icon.svg file is already referenced in the HTML and modern browsers will use it automatically.
